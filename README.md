# `<programming_language>`

A programming language similar to Python.

## `"Hello, world!"`
```
function run() {
    output("Hello, world!")
}
```

## Operators
### Assignment
* `:` - Used to assign values to variables

### Arithmetic
* `+` - Addition
* `-` - Substraction
* `*` - Multiplication
* `/` - Division
* `%` - Modulus operation
* `^` - Exponentiation

### Comparison
* `>` - Greater than
* `>=` - Greater than or equal to
* `<` - Less than
* `<=` - Less than or equal to
* `==` - Equal to

## Keywords
* `function` - Keyword for function declaration
* `run()` - The entry point of every program
* `output()` - The print function
* `input()` - The user input function
* `if` - If statement keyword
* `elsif` - Else if statement keyword
* `else` - Else statement keyword
* `for` - For loop keyword
* `while` - While loop keyword
